**추천: 이미지 선택 시 나타나는 플로팅 버튼**

고객이 이미지를 캔버스에 올리고 클릭하면, 이미지 위에 작은 버튼이 하나 뜹니다.
```

```
클릭하면 오른쪽 사이드바 또는 모달창으로 스타일 선택 패널이 열립니다.

- 기존 UI를 건드리지 않음

- 자연스러운 흐름 (업로드 → 변환 → 캔버스 배치)
